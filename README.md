# Student Performance Prediction System

## 📌 Overview
This project uses Machine Learning to predict student marks based on study hours and classify whether a student will pass or fail.

## 🧠 Problem Statement
To analyze how study hours impact student performance and build predictive models using regression and classification.

## ⚙️ Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## 📊 Models Used
- Linear Regression (for marks prediction)
- Logistic Regression (for Pass/Fail classification)

## 📈 Workflow
1. Data creation and preprocessing  
2. Train-test split  
3. Model training  
4. Prediction  
5. Evaluation (MAE & Accuracy)

## 📉 Results
- Linear Regression predicts marks based on study hours
- Logistic Regression classifies students as Pass/Fail
- Model evaluated using MAE and Accuracy Score

## 📌 Sample Output
- Prediction for 6 hours study: ~60 marks  
- Pass/Fail probability shown using logistic model
